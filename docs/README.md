Project Documentation
=====================

This folder contains documentation for HackMyResume. Please start with the
following resources:

- `USAGE.md` - Quickstart, CLI examples, programmatic examples, and testing.
- `API.md` - Programmatic API reference and samples.
- `FEATURES.md` - A condensed list of supported features.
- `GRUNT_REMOVAL_PLAN.md` - The plan and checklist for removing Grunt.

If you'd like to add or expand docs, please submit a PR targeting `master`.
